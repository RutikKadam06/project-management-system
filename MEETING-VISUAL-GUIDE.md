## Calendar Views

### 1. Month View (Default)
```
Shows: Entire month at a glance
Best for: Long-term planning
Features:
  - See all meetings for the month
  - Click date to create meeting
  - Click event to edit
  - Color-coded events
```

### 2. Week View
```
Shows: 7 days with time slots
Best for: Weekly planning
Features:
  - Hour-by-hour schedule
  - See meeting times clearly
  - Better for busy weeks
  - Time slot selection
```

### 3. Day View
```
Shows: Single day with all hours
Best for: Daily schedule
Features:
  - Detailed hourly view
  - Perfect for busy days
  - See all meeting details
  - Easy time management
```

---

## User Interactions

### Creating a Meeting
```
1. Click "+ Schedule Meeting" button
   OR
   Click on a date in calendar

2. Modal opens with form

3. Fill required fields:
   - Title
   - Start Date & Time
   - Duration
   - Participants

4. Click "Save Meeting"

5. Meeting appears on calendar
   Participants receive notifications
```

### Editing a Meeting
```
1. Click on meeting event in calendar

2. Modal opens with meeting details

3. Modify fields as needed

4. Click "Save Meeting"

5. Meeting updated on calendar
   Participants notified of changes
```

### Deleting a Meeting
```
1. Click on meeting event

2. Modal opens

3. Click "Delete" button

4. Confirm deletion

5. Meeting removed from calendar
```
